# Прототипирование интерфейсов

## Цель: закрепить полученные знания, научиться прототипировать интерфейс, используя основные принципы.

### Задача: 
1. Создайте в Figma для iOS прототип одного из экранов приложения банка.
2. Создайте в Figma для Android прототип главного экрана приложения по учёту расходов.
3. Доработайте экран приложения по учёту расходов, сделав прототип кликабельным.

### Решение
Прежде, чем приступить к выполнению задания, ознакомился с уже существующими приложениями банков, а также с программами по учету расходов. Далее выделил компоненты, которые должны присутствовать на интерфейсе. С применением инструмента Figma создал:
- прототип экрана истории операций в банковском приложении для [IOS](https://www.figma.com/proto/I3Yfkb8VEpcxrx91jNobDA/Untitled?type=design&node-id=12-300&scaling=scale-down&page-id=12%3A5);
- кликабельный прототип главного экрана приложения по учету расходов для [OS Android](https://www.figma.com/proto/I3Yfkb8VEpcxrx91jNobDA/Untitled?type=design&node-id=114-224&t=OlOLJrL9N8L3hEii-0&scaling=scale-down&page-id=71%3A79&starting-point-node-id=71%3A80)
